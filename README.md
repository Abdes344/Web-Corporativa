# Web-Corporativa

# T02 – Proposta de pàgina corporativa FoodLogistic S.A.

## 📌 Descripció del projecte

Aquest projecte consisteix en la creació d'una proposta de nova pàgina web corporativa per a l’empresa FoodLogistic S.A.

L’objectiu principal és millorar la seva presència digital, ja que la seva web actual es troba desactualitzada tant a nivell estètic com tècnic, i no compleix amb la normativa vigent (LOPDGDD i LSSI-CE).

Aquesta proposta inclou:

* Una nova estructura web moderna
* Publicació real mitjançant GitHub Pages
* Integració d’un sistema d’analítica web (StatCounter)
